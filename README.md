# Site de Lista de Compras - Quicklist

## Descrição

Este é um site simples e intuitivo para criar e gerenciar listas de compras. Desenvolvido com HTML, CSS e JavaScript puro, permite adicionar itens à lista, marcar como selecionados e remover itens indesejados. Ideal para compras semanais ou diárias.

## Funcionalidades

- **Adicionar itens**: Digite o nome do item no campo de entrada e clique em "Adicionar item" ou pressione Enter.
- **Marcar itens**: Clique no botão "Selecione" para marcar um item como selecionado (útil para itens já comprados).
- **Remover itens**: Clique no botão "Excluir" para remover um item da lista.
- **Interface responsiva**: Design simples e limpo, otimizado para dispositivos móveis e desktop.

## Tecnologias Utilizadas

- **HTML5**: Estrutura da página.
- **CSS3**: Estilização e layout responsivo.
- **JavaScript (ES6)**: Lógica de interação e manipulação do DOM.

## Como Executar

1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` em qualquer navegador web moderno (Chrome, Firefox, Safari, etc.).
3. Comece a adicionar seus itens de compras!

Não é necessário instalar nenhum servidor ou dependências adicionais, pois é um projeto totalmente estático.
